# 🐍 Snake Puzzle

เกม Snake Puzzle เป็นเกมงูแนว Puzzle ที่ผู้เล่นต้องควบคุมงูให้กินแอปเปิ้ลเพื่อยืดตัวให้ยาวพอ แล้วเข้าประตูมิติเพื่อผ่านด่าน โดยต้องระวังไม่ให้งูตกเหว พัฒนาด้วย Python และ Kivy

---

## 🛠️ เทคโนโลยีที่ใช้

- **Python 3**
- **Kivy** — สำหรับ UI และ Graphics
- **Git** — สำหรับจัดการ version control

---

## ⚙️ วิธีติดตั้งและรัน

### 1. ติดตั้ง Python
ดาวน์โหลดและติดตั้ง Python 3 จาก [python.org](https://www.python.org/)

### 2. ติดตั้ง Kivy
```bash
pip install kivy
```

### 3. Clone โปรเจค
```bash
git clone <repository-url>
cd SnakePuzzle
```

### 4. รันเกม
```bash
python main.py
```

## 🎮 วิธีเล่น

| ปุ่ม | การกระทำ |
|------|----------|
| ⬆️ Arrow Up | เดินขึ้น |
| ⬇️ Arrow Down | เดินลง |

### เป้าหมาย
1. กิน 🍎 แอปเปิ้ลให้ครบเพื่อยืดตัวงู
2. เข้า 🌀 ประตูมิติเพื่อผ่านด่าน
3. ระวังอย่าให้งูตกเหว!

---

## ✨ Features

- 🐍 **งูเดินและตกเหว** — งูจะได้รับแรงโน้มถ่วง ถ้าไม่มีพื้นรองจะตกเหวและ Game Over
- 🍎 **กินแอปเปิ้ลเพื่อยืดตัว** — กินแอปเปิ้ลแล้วตัวงูจะยาวขึ้น
- 🌀 **ประตูมิติ** — เข้าประตูมิติเพื่อไปด่านถัดไป
- 🪨 **หินดันได้** — สามารถดันก้อนหินเพื่อสร้างพื้นยืนได้
- 🗺️ **5 ด่าน** — แต่ละด่านมี background และ layout แตกต่างกัน
